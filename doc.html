Implementace

Implementace je v jazyce python s pomocí mikroframeworku Flask. Všechny případy užití jsou implementovány v souboru app/routes.py.

Databáze

Databáze je typu MySQL. Pro samotnou práci jsem využila ORM Peewee. Inicializace databáze se provede python scriptem create_db.py. Databázové modely se nachází v souboru app/models.py

Instalace

Instalace na server:

  • Připojit se na webový portál pythonanywhere.com
  • Přes bash konzoli v portálu stáhnout git repozitář s projektem
  • Přes záložku web vytvořit webovou aplikaci a odkázat na soubory stáhnuté v předchozím kroku
  • Webová aplikace je přístupná

Softwarové požadavky

Python 3.8, python knihovny napsané v souboru requirements.txt

Databáze

Je nutné mít nainstalované MySQL. A příslušný MYSQL Driver pro python (je napsané v requirements) např. mysqlclient.
Veškeré ostatní kroky zajistí python script create_db.py. Stačí ho spusit pomocí python3 create_db.py

Známé problémy

Z časových důvodů nebyla vyřešena editace vozidel, požadavků a záznamů o údržbě (jejich přidávání však funguje). Zobrazování závad správcem. Ukázkové video.
